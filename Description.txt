
Программа запуска/остановки сервисов Windows ServiceServer 1.0
Автор: Дупляк Анна
Дата: 28.11.2013


Здадание:

Реализуйте, пожалуйста, программу для управления сервисами виндовс (старт, стоп).
 
Условия: Функциональность по управлению сервисами реализовать в COM сервере (С++), пользовательский интерфейс на C#.
Операции с сервисами должны быть асинхронны,  уведомления посылать посредством событий COM.
 
Программа должна быть стабильна, не должна «зависать» и «падать» при любых сценариях работы.
Результат выслать в виде проекта VS любой версии, компилироваться и запускаться программа должна без дополнительный настроек.

Описание проекта:

ServiceServer - программа для запуска или остоновки Windows сервисов. 


Журнал сервера находится в папке \jobTest\ServiceServer\SWServerDLG\bin\Debug\log_0.log


Проект имеет следующие зависимости от внешних библиотек:

-boost threads
-boost log 2.0

Если данные библиотеки уже установлены на вашем компьютере, то вам следует изменить путь к ним в свойствах проекта

Properties->C/C++->General->Additional Include Directories
Properties->Linker->General->Additional Library Directories

Программа предназначена к использованию с правами администратора.

Чтобы построить проект запустите VS с правами администратора и скомпилируйте ServiceServer
Затем постройте основной проект.
